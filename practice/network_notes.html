Information about Networks

Vocabulary

Protocol
Set of rules that all computers and people have to follow while on the internet
Packet
Part of a file that has been torn up to be the correct size for transport
Header
Information about destination and source of the packet
Payload
Actual file or data that is being transported
Router
Device that connects Everyone's computer to the internet
Routing Table
Rules that help tell the Router how to send the packets

Network Problems Experienced

Problem: The computers in the middle were being overwhelmed with papers and could not get them all out.
Solution: Each computer had to wait 30 seconds before sending another packet into the network, to make sure no one was overwhelmed.


Problem: The packets were coming out of order and it was difficult for some computers to put the pieces together
Solution: Each packet had a written note on it saying where it goes in regards to the whole file.


Problem: Some packets were disappering, which is called dropped.
Solution: This problem will keep happening, but it is lessened when all the computers are working together to deliver the message to its destination.
