Important Things to know about in Vim

Here is some information in Vim that can help you in ITD 110

How to access your index.html file

  1. To start, you need to open the terminal
  2. enter your website folder (cd website)
  3. open index.html (type vim index.html)

What do you type into vim?

Start by typing this code into vim

<!DOCTYPE html>
<html>
	<head>
		<title>Homepage</title>
	</head>
	<body>
		<p>Welcome to my first HTML page.</p>
	</body>
</html>

What is so important about this code?

  • starting code for all html documents
  • has all the important things needed in a html document
  • allows for addtions in html document afterwards

Now you can add more tags! Here are a few examples

The <p> tag
Paragraph tag; allows you to write down text in a paragraph
The <q> tag
Quotation tag; puts quotes on to text
The <blockquote> tag
Blockquote tag; tag used for bigger quotations that take up more than one line
The <hr> tag
Horizontal Rules tag; puts a horizontal line on the webpage

You can also add attributes, attributes are code that add on to tags

The <mark> attribute
Highlight attribute; highlights the text
The <strong> attribute
Bold attribute;bolds the text
The <u> attribute
Underline attribute; underlines the text
The <em> attribute
Italic attribute; puts text into italics
The <del> attribute
Strike attribute; puts a line through the text

How to save your vim file

Press :wq if you want to save it, if you do not want to save, press :q!

How can I check that my code is correct?

If you want to check your code, you can add a validator.
Here is the code needed to add said validator

<script src="js/validate.js"></script>

Note: this code will work if you structured your files the exact way taught in Web Design Class


Now that you have this code, once you look at your website, you will see either
HTML/CSS Valid!
Or
HTML/CSS NOT Valid!

If your code is not vaild, It will likely tell you why, so you can go in and try to fix it
